    Matematica e estatistica

    Amostragem

    • Aleatória Simples

      • É a amostragem mais comum, a que coleta elementos aleatórios dentro de um todo.

    • Sistemática

      • É o tipo de amostra que seleciona os elementos de uma forma padronizada, como por exemplo um intervalo entre um número e outro.

    • Estratificada

      • Já a estratificada é quando uma população é dividida em segmentos menores, de forma que é pego uma seleção proporcional ao que a divisão representa.

    Escala de medição

    • Nominal

      • A escala nominal é uma maneira simples de classificar variáveis por meio de nomes ou categorias, sem uma ordem específica ou valores quantitativos associados. Ela é amplamente utilizada em pesquisas e estudos para agrupar e analisar dados em diferentes categorias.

    • Ordinal

      • Diferentemente da anterior esta tem uma função de categorizar, uma ordem e apresenta valores, apesar de não ter uma diferença quantitativa estabelecida entre eles.

    • Intervalar

      • Como o próprio nome ja diz é uma escala que resolve o problema da anterior, ela possui um intervalo melhor definido, o que possibilita uma diferença quantitativa precisa entre eles.

    • Razão

      • Esta faz tudo que a anterior e também pode acomodar o valor zero como valor absoluto, podendo também realizar cálculos com relação entre os valores.

    Tendencia Central

    • Moda

      • Moda é algo que nos é ensinado desde o fundamental como um conjunto de dados, que são definidos a partir de suas aparições dentro de um todo, podendo assim ser classificado como modal quando a maior moda é um elemento, bimodal com dois, polimodal com 3 ou mais e amodal com nenhum.

    • Mediana

      • Mediana é dada como o valor central dentro de um conjunto de elemento, no caso de ele ser impar, quando este conjunto é par é necessário coletar a média dos produtos mais próximos ao centro.

    • Média

      • Média é o resultado da conta em que se soma x quantidade de elementos e este valor é divido pela quantidade de elementos, no caso x.

    • Percentil

      • Percentil é uma medida aplicada em um conjunto de elementos, no qual ira de 0% até 100%, e quanto mais elementos, mais fragmentado sera o valor do percentil.

    • Quartil

      • Quartil tem a mesma lógica de ser porcentagem de um todo, porém ele só se aplica quando esta mesma for 1/4, 2/4 e 3/4, dividindo assim o conjunto em quatro partes.

    Medidas de dispersão

    Medidas de dispersão pode ser dito como um valor para o grau de variabilidade dos elementos de um conjunto

    • Amplitude

      • Ela representa a diferença entre o maior e o menor valor em um conjunto de dados, de forma que, ela nos dá uma ideia de quão espalhados ou concentrados os valores estão. Quanto maior a amplitude, maior é a dispersão dos dados.

    • Variância

      • A variância é uma medida que nos ajuda a entender o quão diferentes os valores de um conjunto de dados estão em relação à média. Ela é calculada encontrando a média dos quadrados das diferenças entre cada valor e a média do conjunto.

    • Desvio Padrão

      • O desvio padrão é uma medida que mostra o quão espalhados estão os valores em relação à média. Se o desvio padrão for grande, os valores estão mais afastados da média. Se o desvio padrão for pequeno, os valores estão mais próximos da média.

    Teste de Hipóteses

    Os testes de hipóteses são uma forma estatística de avaliar se uma afirmação sobre uma população é válida ou não com base em uma amostra de dados. Caso deseje saber mais Clique aqui ou procure por Teste Kolmogorov-Smirnov.

    Inferência

    Inferência é o processo de fazer conclusões sobre uma população com base em informações limitadas de uma amostra. Isso envolve estimar valores desconhecidos da população e fazer testes para validar afirmações.

    Análise

    • Regressão

      • A análise de regressão é uma ferramenta estatística que nos ajuda a entender como uma variável está relacionada a outras variáveis. Ela usa equações matemáticas para encontrar padrões e fazer previsões.

    • Séries Temporais

      • A análise de séries temporais é o estudo de conjuntos de dados coletados ao longo do tempo. Ela nos ajuda a identificar padrões e tendências nesses dados para fazer previsões futuras.

    • Correlação e Associação

      • A análise de correlação busca medir a força e a direção da relação linear entre duas variáveis. Ela utiliza o coeficiente de correlação para expressar essa relação, que varia de -1 a 1. Por outro lado, a análise de associação é usada para determinar se existe uma relação estatisticamente significativa entre duas variáveis categóricas.

    Limites

    Limite de uma função refere-se ao valor para o qual uma função se aproxima quando a variável de entrada se aproxima de um determinado ponto. É representado como f(x) à medida que x se aproxima de um valor específico, como x → a.

    Por exemplo, na função f(x) = 2x + 2 e quisermos calcular o limite quando x se aproxima de 4, podemos substituir x por 4 na função:

    f(4) = 2(4) + 2 = 10.

    Ou seja neste caso, o limite da função f(x) quando x se aproxima de 4 é 10. Isso significa que a função se aproxima de 10 à medida que x se aproxima de 4.

    Derivadas

    Derivadas são uma medida da taxa de variação de uma função em relação à sua variável de entrada, ela pode fornecer informações sobre a inclinação da curva da função, a taxa de variação instantânea ou a taxa de crescimento da função em um determinado ponto, para saber mais entre no nosso colab apertando aqui.

    Integrais

    Integrais são usadas para calcular a área sob uma curva ou o acumulado de uma quantidade ao longo de um intervalo. Elas são calculadas usando limites.

    Para calcular integral basta:

    1. Escolha a função que representa a curva ou a quantidade que você deseja integrar.
    2. Defina o intervalo no qual você deseja calcular a integral.
    3. Aproxime a área sob a curva em cada subintervalo usando retângulos ou trapézios.
    4. Some todas as áreas aproximadas para obter a integral aproximada.

    Por exemplo, se quisermos calcular a integral da função f(x) = 2x no intervalo de 0 a 4, podemos dividir o intervalo em subintervalos menores e aproximar a área sob a curva usando retângulos.

    Ao somar as áreas dos retângulos, obtemos uma estimativa da integral. Nesse caso, a integral de f(x) no intervalo de 0 a 4 é aproximadamente igual a 16.
